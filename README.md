# Sound Flow Korean

A Vite React test app for Korean pronunciation practice and English intonation visualization.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

GitHub Pages deployment is configured in `.github/workflows/deploy.yml`. The Vite `base` is `/`, which is correct for a custom domain at the domain root such as `electstat.kr` or `electstat.com`.
